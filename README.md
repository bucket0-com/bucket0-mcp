# bucket0-mcp

The remote server for **AgentBucket**. Give your AI agent a file system that remembers.

Add one URL to Claude, ChatGPT, or any MCP-compatible assistant, and your agent can save files, recall them by meaning, and read them back across sessions.

## Connect

In your assistant, add a custom connector and point it at:

```
https://mcp.bucket0.com/mcp
```

Click **Connect**, sign in with your Bucket0 account, and approve. No keys to paste. You can disconnect any time from your Bucket0 dashboard.

## What your agent can do

- Save files: research, notes, documents, data, code
- Recall earlier files by meaning, not just by name
- Read, list, and organize what it has saved
- Delete on request

Recall by meaning is available on paid Bucket0 plans.

## Using a local assistant?

For Claude Code, Claude Desktop, or Cursor, use the [`agentbucket`](https://www.npmjs.com/package/agentbucket) package instead.
